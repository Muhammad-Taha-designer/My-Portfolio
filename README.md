## 💡 The Design System & Aesthetic

To match the serious, immersive tone of the project, the visual language utilizes a dark, atmospheric layout contrasted with vibrant accent elements to draw the eye to critical actions.

| Element | Specification | Visual Reference / Description |
| :--- | :--- | :--- |
| **Primary Color** | `#000000` / Deep Dark Gray | Establishes a premium, gritty cinematic backdrop. |
| **Accent Glow** | `[e.g., #00D2FF / Neon Cyan]` | Used exclusively for high-priority CTA buttons and active states. |
| **Typography** | `[e.g., Space Grotesk / Inter]` | Bold, geometric headers for a high-tech feel; highly readable body text. |

## 🗺️ User Flow & Architecture

Before diving into the high-fidelity visuals, the core focus was creating a friction-free user path that gets the user to key destinations in under 3 clicks.
